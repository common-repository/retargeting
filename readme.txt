== Description ==


An easy way to place retargeting pixels and code on your Wordpress site.

###What you get when you use the Retargeting plugin

*  Put retargeting pixel or code on any post or page.
*  Place retargeting pixels and code in the head and footer.


###Add retargeting pixel to any post or page

Whether you are writing a post about your company or another company, you can include a well formatted retargeting pixel.

###Quick Start Guide

* [Retargeting plugin quick start guide at thinklandingpages.com] (http://www.thinklandingpages.com/retargeting-plugin/)

###Retargeting Pixel Placement 

Add as many retargeting pixel to your site as you like.  



== Installation ==


1. Upload `Retargeting` to the `/wp-content/retargeting/` directory
1. Activate the retargeting through the 'Retargeting' menu in WordPress
1. Click **Retargeting** on the admin menu to enable and set your options.


== Frequently Asked Questions ==

= Do I have to know how to program or design? =

No.  The retargeting pixel placement does the programming and design.

= Is there a limit to the number of retargeting pixel I can create? =
No, unlimited retargeting pixels and codes can be created.

= Can the retargeting pixel be put on my homepage? =
Yes, you can put retargeting pixel on your homepage.

= Can I put retargeting pixel placement on any post or page? =
Yes, use the retargeting pixel on your pages and posts.
